Users\Cole-Williams\Documents\Projects\RASPICO-8

  • The RASPICO-8 is a miniaturized arcade cabinet that can fit on a desk, running of the PICO-8 operating system
  • This operating system supports 16 colors, 128x128 pixels, and a maxiumum size of 32kb
  • For input, the RASPICO-8 uses 3 backlit buttons, and an 8 directional joysticl (though a keyboard and mouse can be used to play certain games)
  • The device works from a RASPBERRY-PI4, and the USB inputs are read by a program called AntiMicroX
  • AntiMicroX converts controller input into keyboard based commands, based on a configuration file designed for this
