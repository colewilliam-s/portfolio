The RASPICO-8 is a miniaturized arcade cabinet that can fit on a desk, and runs on the
PICO-8 operating system that supports 16 colors, 128x128 pixels, and a maximum size of 32kb
For input, the RASPICO-8 has 3 backlit buttons, and an 8 directional joystick
(Though a keyboard and mouse can be used to play certain games)

The device works from a RASPBERRY-PI4, and the USB inputs are read by a program called AntiMicroX,
Where it converts the controller input into keyboard based commands, based on a configuration file designed for this
